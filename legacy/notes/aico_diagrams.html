The Boot Sequence
From cold silicon to locked runtime in under 10 seconds
01
BIG BANG
Atomic activation. Bootstrap loads monolithic binary from OpROM or USB. No service dependencies. Single-shot integrity verification and handoff.
ATOMIC OPROM NO DEPS
02
SHIBBOLETH
Hardware proof-of-control. Bit-bang Fan Header PWM pin to transmit ASCII "READY" at 9600 baud. Proves microsecond-level CPU authority over the chipset. Failure triggers immediate safe shutdown — no retry.
FAN HEADER 9600 BAUD FAIL = SHUTDOWN
03
NEST BUILDING
Optimization Storm. JIT-compile drivers for CPU, GPU, and Memory directly from hardware register maps. Each subsystem follows the 3-Upgrade Rule: Draft → Refine → Polish → LOCK. Duration target: under 10 seconds.
JIT SYNTHESIS 3-UPGRADE LOCK <10s WRITE ACCESS REVOKED
04
GESTATION
Format Soul. Map memory. Build User Vault. Establish Introspection baseline. Serial log emitted via Fan Header throughout. Mascot initialized in outer zone.
MEMORY MAP USER VAULT SERIAL LOG
05
FIRST CONTACT
Cold boot complete. Generative UI compiled to framebuffer. Local interface only. System ready for cognitive operation within the locked Nest.
